# OpenOccupancy Feature Extraction Modification

本文档说明在原始 **OpenOccupancy** 项目基础上，为实现 **3D voxel feature 提取（FC 层之前）**所做的修改。

目标：  
提取 **OccNet 在 Head 之前的 3D 体素特征**，用于测试或下游任务。

---

# 1. 修改目的

原始 OpenOccupancy 的 `OccNet` 在推理阶段会执行：

```

feature extraction → occupancy head → semantic prediction

```

但在本项目中，我们仅需要：

```

feature extraction → 输出 voxel feature

```

即 **获取 FC 层之前的 3D 特征表示**。

---

# 2. 模型结构修改

修改文件：

```

projects/occ_plugin/occupancy/detectors/occnet.py

````

主要修改如下。

---

## 2.1 关闭训练 Head

原始 `forward_pts_train()` 会调用 `pts_bbox_head` 进行预测并计算损失。

修改后：

```python
def forward_pts_train(self, *args, **kwargs):
    losses = dict()
    return losses
````

作用：

* 不调用任何 Head
* 不计算 occupancy loss
* 只保留特征提取部分

---

## 2.2 修改 forward_train()

原始流程：

```
extract_feat
↓
pts_bbox_head
↓
loss
```

修改后仅执行：

```python
voxel_feats, img_feats, pts_feats, depth = self.extract_feat(
    points, img=img_inputs, img_metas=img_metas
)
```

并返回一个 dummy loss：

```python
dummy_loss = torch.tensor(0.0, device=points[0].device, requires_grad=True)
losses['dummy_loss'] = dummy_loss
```

作用：

* 保持训练流程兼容
* 避免调用预测 head

---

## 2.3 修改 simple_test()

原始 `simple_test()` 会执行：

```
extract_feat → head → prediction
```

修改后：

```python
def simple_test(self, img_metas, img=None, points=None, **kwargs):
    voxel_feats, img_feats, pts_feats, depth = self.extract_feat(points, img=img, img_metas=img_metas)
    return voxel_feats
```

推理阶段直接返回：

```
voxel_feats
```

即：

```
3D voxel feature (before FC layer)
```

# 3. 新增特征提取脚本

新增脚本：

```
exfeature.py
```

功能：

* 加载 OccNet 模型
* 读取 nuScenes 数据
* 调用 `extract_feat()`
* 保存 voxel 特征

核心代码：

```python
voxel_feats, img_feats, pts_feats, depth = model.extract_feat(
    points=points,
    img=img_inputs,
    img_metas=img_metas
)

feat_3d = voxel_feats[0].cpu().numpy()
```

保存为：

```
npy file
```

---

# 4. 输出特征说明

保存格式：

```
nusc_fc_features/
    sample_token.npy
```

特征 shape：

```
(B, C, X, Y, Z)

example:
(1, 256, 128, 128, 10)
```

含义：

| 维度 | 说明                |
| -- | ----------------- |
| B  | batch size        |
| C  | feature channel   |
| X  | voxel grid x      |
| Y  | voxel grid y      |
| Z  | voxel grid height |

该特征为：

```
Occ Encoder 输出
= FC 层之前的 3D 表征
```

---

# 5. 特征提取流程

整体 pipeline：

```
Camera Images + LiDAR
        │
        ▼
Image Backbone (ResNet50)
        │
        ▼
View Transformer (Lift-Splat-Shoot)
        │
        ▼
LiDAR Voxel Encoder
        │
        ▼
Fusion (VisFuser)
        │
        ▼
3D Encoder (ResNet3D + FPN3D)
        │
        ▼
voxel_feats
```

输出即为：

```
3D voxel feature
```

---

# 6. 使用方法

运行：

```
python exfeature.py
```

脚本会：

1. 加载模型
2. 构建 nuScenes dataset
3. 提取特征
4. 保存 `.npy` 文件

---

# 7. 注意事项

1. 默认只用于 **特征提取测试**
2. 不支持 occupancy prediction
3. 不计算任何训练 loss
4. 输出特征体积较大（约 100MB / frame）

---

# 8. 总结

本修改版本的 OpenOccupancy：

* 移除了 occupancy prediction head
* 保留完整 **multi-modal feature extraction**
* 支持直接输出 **3D voxel feature**
